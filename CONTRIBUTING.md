# Contributing to Draggy Encoder

Thank you for your interest in contributing to Draggy Encoder!

## How to Contribute

1.  **Report Bugs**: If you find a bug, please open an issue describing it.
2.  **Suggest Features**: Have an idea for a new feature? Open an issue to discuss it.
3.  **Pull Requests**:
    - Fork the repository.
    - Create a new branch for your feature or bug fix.
    - Make your changes.
    - Ensure your code follows the project's style.
    - Submit a pull request.
